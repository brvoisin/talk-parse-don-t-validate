La fonction est partielle :
toutes les entrées possibles ne sont pas prises en compte.

Voici une fonction qui retourne le premier élément d'une séquence.

Dans ce cas très simple, on voit tout de suite qu'une séquence vide va lever une exception.

La fonction est partielle…

Le langage me permet de le faire et surtout l'analyse statique des types aussi. (mypy)

Fonction totale

Toutes les entrées possibles sont prises en compte.

Il y a deux façons de passer la fonction head de partielle à totale.

Solution 1 : gérer la séquence vide


            from collections.abc import Sequence

            def head[T](seq: Sequence[T]) -> T | None:
                if not seq:
                    return None
                return seq[0]
          

              print(head([]))
              print(head([1, 2, 3]))
              print(head([None]))
            

              None
              1
              None

Solution 1 : gérer la séquence vide


            from collections.abc import Sequence
            from typing import NewType

            NothingType = NewType("NothingType", str)
            Nothing = NothingType("Nothing")

            def head[T](seq: Sequence[T]) -> T | NothingType:
                if not seq:
                    return Nothing
                return seq[0]
          

              print(head([]))
              print(head([1, 2, 3]))
              print(head([None]))
            

              Nothing
              1
              None
