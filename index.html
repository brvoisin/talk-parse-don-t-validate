Solution 1 : gérer la séquence vide


            def head[T](a: Sequence[T]) -> T | None:
                if not a:
                    return None
                return a[0]
          

              print(head([]))
              print(head([1, 2, 3]))
              print(head([None]))
            

              None
              1
              None

Solution 1 : gérer la séquence vide


            from typing import NewType

            NothingType = NewType("NothingType", str)
            Nothing = NothingType("Nothing")

            def head[T](a: Sequence[T]) -> T | NothingType:
                if not a:
                    return Nothing
                return a[0]
          

              print(head([]))
              print(head([1, 2, 3]))
              print(head([None]))
            

              Nothing
              1
              None
